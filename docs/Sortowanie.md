# Sortowanie

- Porządkowanie zbioru na podstawie kolumny
```sql
SELECT * FROM customer
ORDER BY first_name
```

- Porządkowanie zbioru na podstawie kilku kolumn
```sql
SELECT * FROM customer
ORDER BY first_name, last_name
```

- Porządkowanie malejąco
```sql
SELECT * FROM customer
ORDER BY first_name DESC
```
